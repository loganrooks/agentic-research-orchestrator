# Runner Guide: Gemini Deep Research (v1)

## Expected output shape
Report-first, citation-heavy.

## Why to avoid rigid formatting demands
Over-constraining output (e.g., requiring multiple separate artifacts) can cause partial compliance and loss of nuance.

## What to request
- One primary report with citations.
- Prefer JSON registers; allow table fallback.
- Require explicit contradictions + assumptions + probes.
- Include “Residuals / Open Questions”.
